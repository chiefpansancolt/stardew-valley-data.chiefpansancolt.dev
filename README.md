# Stardew Valley Data - Documentation Site

Documentation site for the [stardew-valley-data](https://github.com/chiefpansancolt/stardew-valley-data) NPM package.

Built with [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com), and [Markdoc](https://markdoc.io).

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```
